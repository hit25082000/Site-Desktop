Senior Automation Engineer

San Francisco, CA (Hybrid)

Full-Time

Build production automation systems that eliminate manual work for enterprise clients. You'll design workflows, write integrations, and deploy systems that process millions of transactions monthly.

Apply now

01

What you'll do

  • Design and implement automation workflows using Python, JavaScript, and API integrations

  • Build custom integrations with enterprise systems (Salesforce, NetSuite, SAP, etc.)

  • Deploy production systems with 99.5%+ uptime requirements

  • Collaborate with clients to understand processes and design solutions

  • Write maintainable code and comprehensive documentation

  • Debug and optimize existing automation systems

02

What we're looking for

  • 5+ years building production software systems

  • Expert in Python or JavaScript/TypeScript

  • Experience with API integrations and webhook architectures

  • Comfortable working directly with clients on technical requirements

  • Track record of shipping systems that actually get used

  • Bonus: Experience with RPA tools, workflow automation platforms, or ETL systems

03

What we offer

  • $140K-180K base + equity

  • Remote-first with quarterly team offsites

  • Health, dental, vision insurance

  • Unlimited PTO (actually use it)

  • $3K/year learning budget

  • Latest MacBook Pro + equipment stipend
